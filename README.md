# BNS-GameJam

Siv3Dでアクション対戦ゲームを作りました

# ゲームタイトル

鍋パニック！逆襲の具材たち

# 機能

1P：WASDキー + Q or コントローラ

2P：↑↓←→キー + / or コントローラ

# 使用法

以下のURLからダウンロードしてください

[ダウンロードURL]
