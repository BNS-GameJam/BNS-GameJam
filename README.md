# BNS-GameJam

Siv3Dでアクション対戦ゲームを作りました

# ゲームタイトル

鍋パニック！逆襲の具材たち

# 機能
- 操作方法
  - キーボード操作
    - 1P : WASD移動,　Qアクションを起こす
    - 2P : ↑↓←→移動,　/アクションを起こす
  - コントローラー操作
    - 十時ボタン移動,　Ⓐ️orⓍアクションを起こす
